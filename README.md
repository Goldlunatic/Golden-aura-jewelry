
# Golden Aura Jewelry Website

This is a ready-to-host version of the Golden Aura Jewelry website.

## Hosting Instructions

### GitHub Pages
1. Create a GitHub repository and upload all files.
2. Go to Settings → Pages → Source → Choose `main branch / root`.
3. Your website will be live at: `https://yourusername.github.io/repositoryname/`

### Netlify
1. Sign up at https://www.netlify.com/
2. Drag & drop the contents of this ZIP onto Netlify dashboard.
3. Netlify will deploy it instantly with a free URL.
